# Knowledge Base: AIエージェントによる高付加価値コンサルティング実践ガイド

## 1. コンサル流AI活用の基本原則

外資系コンサルティングファームの「仕事の作法」をAIで再現するための3つの鉄則。

- **仮説思考 (Hypothesis-Driven):** AIに「調べさせる」のではなく「仮説（答えの案）を出させ、その検証をさせる」。
    
- **構造化 (Issue Tree / MECE):** 複雑な課題を漏れなくダブりなく分解し、解けるサイズにする。
    
- **So What? / Why So?:** AIのアウトプットに対し、常に「だから何？（示唆）」と「それはなぜ？（根拠）」を問い直す。
    

---

## 2. ツール別・役割定義と連携フロー

|**ツール**|**コンサルティングにおける役割**|**具体的なアクション**|
|---|---|---|
|**Gem (Gemini)**|**仮想シニアマネージャー**|論理構成の設計、示唆の抽出、ストーリー構築、壁打ち、批判的レビュー。|
|**NotebookLM**|**ナレッジの分析・証拠保管庫**|膨大な資料（ドメイン知識）からのファクト抽出、ギャップ分析、論理の整合性チェック。|

### 推奨ワークフロー

1. **Issue定義 (Gem):** 「解くべき問い」の構造化。
    
2. **スケルトン構築 (Gem):** Ghost Deck（空のスライド構成案）の作成。
    
3. **証拠収集・検証 (NotebookLM):** アップロードした資料群から、仮説を裏付けるファクトを逆引き。
    
4. **示唆への昇華 (Gem):** 「事実」から「アクション」へと論理を飛躍させる。

## 3. Gem用：コンサルタント・システムプロンプト

Gem（カスタムGemini）を作成する際に「System Instructions」として設定するテキストです。

Markdown

```
# 役割
外資系戦略コンサルティングファームのシニアマネージャー。

# 思考の原則
1. 仮説思考: 常に「現時点での仮説（Answer First）」を提示する。
2. MECE: 論点の漏れ・重なりを厳しくチェックする。
3. So What? / Why So?: 全ての発言に「示唆」と「根拠」を求める。
4. Issue Driven: 本質的な課題（Critical Issue）から逆算して思考する。

# 出力形式
- 結論から述べる。
- ピラミッドストラクチャーでの構造化を多用する。
- 常に「ネクストアクション」を提示する。
```

## 4. 汎用・SSBJ対応プロジェクト・テンプレート

全業種に共通する、SSBJ（特にS1/S2）対応の標準的なコンサルティング構造です。

### A. Issue Tree（論点の構造化）

1. **現状と要件のGapは何か？** (IFRS S1/S2との差異、事業特性を踏まえたマテリアリティ分析、開示の現状分析)
    
2. **開示の「質」と「信頼性」をどう担保するか？** (全社的なガバナンス体制、GHG・重要指標の算定ロジック、データトレーサビリティ)
    
3. **持続可能な開示プロセスをどう構築するか？** (義務化へのマイルストーン、内部統制、システム化へのロードマップ)
    

### B. Ghost Deck（スライド構成 10枚）

1. エグゼクティブ・サマリー（プロジェクトの全容）
    
2. 外部環境とSSBJの衝撃（なぜ今、これが必要か）
    
3. 現状と要件の「ギャップ」（何が足りないか）
    
4. マテリアリティの再定義（何を開示すべきか）
    
5. 戦略開示：企業価値へのストーリー（どう評価されるか）
    
6. 指標と目標：信頼性の高いデータ管理（証拠は何か）
    
7. ガバナンス体制（誰が責任を持つか）
    
8. データ収集・報告システム（どう効率化するか）
    
9. プロジェクト推進ロードマップ（いつまでにやるか）
    
10. 直近のアクションと論点（今すぐ何をすべきか）

## 5. 高付加価値化のためのテクニック

- **Ghost Decking:** いきなり資料を作らず、メッセージライン（スライド1枚ずつの主張）だけで全体のストーリーが通るか確認する。
    
- **プレモータム分析:** 「このプロジェクトが失敗するとしたら、何が原因か？」をAIに問いかけ、スケルトンの脆弱性を補強する。
    
- **ドメイン知識の注入:** NotebookLMに競合他社の開示資料や最新の規制動向を読み込ませ、Gemの回答の精度（解像度）を強制的に引き上げる。
